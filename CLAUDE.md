# unlimited-mcp

Repo de experimentación con coding agents y orquestación de LLMs.

## Carpeta `local/`

Carpeta privada, gitignoreada. Nunca se sube al repo público. Contiene:

- `COMMANDS.md` — referencia de comandos y notas de todos los agentes probados (OpenCode, Hermes, Aider, smolagents, Claude CLI, Codex, jcode…)
- Scripts intermedios y experimentos que no forman parte del repo
- Salidas de comandos y logs de pruebas
- Cualquier fichero temporal o privado

Cuando documentes pruebas, anotes comandos o guardes scripts intermedios, hazlo en `local/`.
